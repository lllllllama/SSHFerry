# MSCP Binary Files

Place the `mscp` binary for your platform in the appropriate directory:

- **Windows**: `win/mscp.exe`
- **Linux**: `linux/mscp`
- **macOS**: `mac/mscp`

## Download MSCP

You can download mscp from: https://github.com/upa/mscp

Or install via package manager:
- Windows: Download from GitHub releases
- Linux: `apt install mscp` or build from source
- macOS: `brew install mscp`

## Alternative

If mscp is in your system PATH, it will be automatically detected.
